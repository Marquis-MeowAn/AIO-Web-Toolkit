# 🧰 Features

## 🔗 Bulk URL Launcher
- Paste and manage multiple URLs  
- Open in tabs, window, or grouped by domain  
- Throttling & delay controls  
- Import / Export support  

## 📝 Sticky Notes
- Rich text editing + images  
- JSON export  
- Local storage  

## 📋 Kanban Board
- Drag-and-drop tasks (SortableJS)  

## 📱 Device Simulator
- iPhone, Android, Tablet, Desktop  
- OS simulation + scaling  

## 🛠 Developer Toolkit
- Case converter  
- Word counter  
- Text cleaner  
- Password generator  

## 🔍 Diff Checker
- Compare text with highlights  

## 🌍 Localization Tools
- Language codes  
- Timezone tools

---

# ⚠️ Limitations
- Limited for very large datasets  
- Performance may degrade with very large datasets
- Some websites cannot be loaded in the simulator due to X-Frame-Options / iframe restrictions
- Data is stored locally (no backend / cloud sync)
- Not optimized for very large datasets (e.g., thousands of URLs or notes)
- Browser-dependent behavior (tab limits, popup restrictions)
- No authentication or multi-user collaboration
